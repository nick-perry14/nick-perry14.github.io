# Compressing Textures

1. Open OpenIV
2. Find the .ytd file 
3. Open .ytd in OpenIV
4. Extract all textures as PNG to a folder
5. Compress PNG files using a compression software (such as [this service](https://compresspng.com) )
6. Replace Interior textures in OpenIV
7. Save new YTD
8. Upload YTD to server
9. Restart Server
